Introdução à Lógica de Programação

Compreendendo os fundamentos para criar soluções eficientes.

O que é Lógica de Programação?

É o conjunto de raciocínios e técnicas usados para resolver problemas de forma lógica e estruturada, usando conceitos como algoritmos.

Conceitos Básicos

  • Variáveis
  • Estruturas de Controle (condicionais e loops)
  • Algoritmos
  • Entrada e Saída

Por Que Aprender Lógica?

A lógica de programação é fundamental para entender qualquer linguagem de programação e desenvolver sistemas eficientes.

  • Facilita o aprendizado de novas linguagens
  • Desenvolve o pensamento crítico
  • Ajuda a resolver problemas do dia a dia

Exemplo Simples

Algoritmo para trocar uma lâmpada:

  1. Desligar o interruptor
  2. Retirar a lâmpada queimada
  3. Colocar a nova lâmpada
  4. Testar o funcionamento

Exemplo de Pseudocódigo

Algoritmo para calcular a média de duas notas: 

INICIO
    LEIA nota1
    LEIA nota2
    soma <- nota1 + nota2
    media <- soma / 2
    SE media >= 7 ENTAO
        ESCREVA "Aprovado"
    SENAO
        ESCREVA "Reprovado"
    FIM_SE
FIM

Conclusão

A lógica de programação é o primeiro passo para se tornar um programador eficaz. Pratique sempre e explore novos desafios!
